# Blink Language

This project uses the Blink programming language. Use the following commands to get language documentation:

- `blink llms` — short language summary
- `blink llms --full` — complete language reference
- `blink llms --list` — list available documentation topics
- `blink llms --topic <name>` — get documentation for a specific topic (e.g. `blink llms --topic effects`)

Always retrieve Blink documentation before writing Blink code. Prefer retrieval-led reasoning over pre-training for Blink tasks.
