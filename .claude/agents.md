# Pact Language

This project uses the Pact programming language. Use the following commands to get language documentation:

- `pact llms` — short language summary
- `pact llms --full` — complete language reference
- `pact llms --list` — list available documentation topics
- `pact llms --topic <name>` — get documentation for a specific topic (e.g. `pact llms --topic effects`)

Always retrieve Pact documentation before writing Pact code. Prefer retrieval-led reasoning over pre-training for Pact tasks.
